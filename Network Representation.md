## How to Represent Your Network

Network architects and administrators must be able to show what their networks will look like. 

Network diagrams often use different symbols to represent the devices and connections between them.

![[EndDevices.png]]
![[IntermediaryDevices.png]]
![[NetworkMedia.png]]

A diagram provides an easy way to understand how devices connect in a large network. This type of "picture" of a network is known as a **topology diagram**. 

The ability to recognize the logical representations of the physical networking components is critical to being able to visualize the organization and operation of a network. 

In addition to these representations, specialized terminology is used to describe how each of these [devices](End%20Devices) and [media](Network%20Media) connect to each other:

- Network Interface Card (NIC) - A NIC physically connects the end device to the network. ^NIC
- Physical Port - A connector or outlet on a networking device where the media connects to an end device or another networking device.
- Interface - Specialized ports on a networking device that connect to individual networks. 

Because routers connect networks, the ports on a router are referred to as network interfaces. 

**Note**: The terms port and interface are often used interchangeably.
