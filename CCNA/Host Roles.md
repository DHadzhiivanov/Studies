## What are Hosts in Networking?

Hosts can be called [end devices](End%20Devices.md) which refers to devices with a designated [IP address](IP%20Addressing.md) for communication.

[Clients](End%20Devices.md) are a **type** of host. Clients have software for requesting and displaying the information obtained from the server, as shown in the figure.

![[Pasted image 20260205185133.png]]

## Examples of Client Software

- Web Browsers like **Firefox** or **Chrome**
- Mail client software like **Microsoft Outlook**
- File explorer software like the **Windows File Explorer**

A single computer can run multiple types of software simultaneously.
