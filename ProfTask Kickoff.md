# 1. Fonteyn Holiday Parks

Design and modernize an infrastructure for the company.

---

# Team Role Descriptions

---

## Architect

Designs the overall IT infrastructure. Decides which technologies to use, how systems connect, and ensures the design is scalable and consistent throughout the project.

**Key tasks**

- Create network and system diagrams
- Define technology stack and design decisions
- Review implementations against the design
- Document architectural choices and rationale

**Works closely with**

- Network Administrator
- Security Officer
- Project Leader
- DevOps Engineer

---

## Project Leader

Owns the overall planning and delivery of the project. Manages deadlines, communicates with the client, and keeps the team aligned on goals and priorities.

**Key tasks**

- Create and maintain the project planning
- Lead client meetings and report on progress
- Resolve blockers and escalate risks
- Ensure deliverables meet requirements

**Works closely with**

- Scrum Master
- Architect
- All roles

---

## Scrum Master

Facilitates the Agile process. Runs daily standups, manages the sprint backlog, and removes anything blocking the team from making progress.

**Key tasks**

- Organise and run daily standups
- Maintain and prioritise the backlog
- Run sprint planning and retrospectives
- Track team velocity and flag delays

**Works closely with**

- Project Leader
- All roles

---

## Network Administrator

Implements and manages the network infrastructure. Translates the Architect's designs into a working, configured environment.

**Key tasks**

- Configure routers, switches and VLANs
- Set up and manage servers and services
- Monitor network performance and uptime
- Troubleshoot connectivity issues

**Works closely with**

- Architect
- DevOps Engineer
- Security Officer

---

## DevOps Engineer

Automates deployment pipelines and manages the server environment. Bridges development and infrastructure to keep deployments smooth and repeatable.

**Key tasks**

- Set up CI/CD pipelines
- Manage containers and deployment environments
- Write automation scripts for infrastructure tasks
- Monitor system health and logs

**Works closely with**

- Architect
- Network Administrator
- Security Officer

---

## Security Officer

Ensures the infrastructure is secure from design through to deployment. Identifies threats, defines security policies, and reviews the work of other roles for vulnerabilities.

**Key tasks**

- Perform threat modelling on the architecture
- Define firewall rules and access policies
- Run security reviews and penetration tests
- Document security measures and incidents

**Works closely with**

- Architect
- Privacy Officer
- Network Administrator
- DevOps Engineer

---

## Privacy Officer

Ensures the project complies with GDPR and data protection regulations. Advises the team on how personal data must be handled, stored, and protected.

**Key tasks**

- Conduct a Privacy Impact Assessment (PIA)
- Define data retention and deletion policies
- Review the architecture for data compliance
- Advise on consent and data access rights

**Works closely with**

- Security Officer
- Architect
- Project Leader
